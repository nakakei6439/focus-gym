# Focus Gym 開発方針・進行計画（統合ドキュメント）

最終更新: 2026-03-12

------------------------------------------------------------------------

# 1. プロジェクト概要

Focus Gym は日常的な目のケアを目的とした視覚トレーニングアプリ。\
ターゲットは **40代女性** を中心としたユーザー。

コンセプト

-   毎日少しだけ目のケア
-   ゲーム感覚で続けられる
-   医療効果を誤認させない設計
-   エビデンスベースのトレーニング

------------------------------------------------------------------------

# 2. 課金モデル（最終仕様）

## 無料

-   遠近ピント切替トレーニング（永久無料）

## 無料トライアル

-   初回起動から **14日間**
-   **全機能使用可能**
-   **カード登録不要**

## トライアル終了後

残りトレーニングはロック

## 有料解放

価格：**300円（買い切り）**

種別：

Non‑Consumable

商品ID

com.focusgym.unlock_all

購入後

-   全トレーニング永久アンロック

### 採用理由

-   サブスクは課金不安が強い
-   40代女性は買い切りを好む傾向
-   トライアル → 購入の導線が自然
